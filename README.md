# EDA_start
 EDA анализа датасета ирисов Фишера | PCA | t-SNE | Кластеризация
О проекте:
Полное исследование данных ирисов Фишера с использованием:
- Разведочного анализа (EDA)
- Визуализации распределений
- PCA и t-SNE для снижения размерности

Библиотеки:
- Python 3
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

Это моя первая подобная работа, надеюсь дальше будет получаться лучше
